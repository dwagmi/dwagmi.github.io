<em>Non-recursive Solution</em>

1. Create a new empty node. 
2. Iterate through both lists simulatenously. 
3. Pick the lower value to add to the new list. 

<iframe src='/leetcode/merge-two-sorted-lists.html' width='100%' height='650px'></iframe> 

<em>Recursive Solution</em>

1. If list1.val < list2.val, then recurse with list1.next and set result to list1.next
2. Otherwise, recurse with list2.next and set result to list2.next
3. If either list1 or list2 are null, then return the non-null list.
